# Prakriti Environmental Club Website

A responsive multi-page website built using HTML, CSS, and JavaScript for an environmental club.  
The website showcases club activities including events, environmental projects, blogs, gallery images, and contact information.

## Features
- Responsive navigation bar
- Events and project showcase
- Blog section
- Image gallery
- Contact form
- Clean and modern UI design

## Technologies Used
- HTML
- CSS
- JavaScript
